# Superpowers OpenCode Contributor Notes

This repository is OpenCode-only. Keep runtime code, docs, tests, and examples focused on OpenCode.

- Do not add Claude, Codex, Cursor, Gemini, Kimi, Pi, Copilot, or Antigravity packaging back into this repository.
- OpenCode integration lives in `.opencode/plugins/superpowers.js`.
- OpenCode docs live in `.opencode/INSTALL.md` and `docs/README.opencode.md`.
- Plugin tests live in `tests/opencode/`.
- Use `apply_patch` for edits and keep changes minimal.
